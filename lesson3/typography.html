Typography

When considering typography we know that readability is the most important as we want to be able to communicate with this site. To combine the visibility and still still have a personable feel we are going with Lily Script for the headings. This will catch the eye yet be easy to read. Slabo 27 is a good choice for readability in the main paragraphs and all basic areas.
